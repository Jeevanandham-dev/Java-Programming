## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Jeevanandham-dev/Java-Programming.git
   
2. Navigate to the folder containing the program.

3. Compile the Java file:
   javac ProgramName.java

4. Run the program:
   java ProgramName
